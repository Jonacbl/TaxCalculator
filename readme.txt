简介：
项目名为TaxCalculator，用于计算个人所得税，以及查看和修改税率表的功能。

特性：
- 开发语言：Java
- 开发环境：IntelliJ IDEA Ultimate 2024.3.4 & Oracle OpenJDK 23.0.2
- 支持命令行执行
- 使用javadoc生成了说明性文档，提供了生成javadoc的脚本docmake.bat
- 提供了运行

项目结构:
TaxCalculator/                  // 根目录
|---- bin/                      // 存放编译生成的中间文件（.class 文件）
|---- docs/                     // 存放 Javadoc 生成的说明性文档
|---- src/                      // 存放源代码文件
|   |---- utils/                // 工具类包
|   |---- Menu.java             // 菜单类
|   |---- TaxInterface.java     // 税务接口
|---- Main.java                 // 项目入口文件
|---- docmake.bat               // 生成文档的批处理脚本
|---- run.bat                   // 运行项目的批处理脚本
